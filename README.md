# Beatrix Organization Config 🛠️

This repository is used to store organization-wide configuration files, profile branding, and community health files for the **Beatrix Programming Language**.

## 📁 Structure
- `/profile`: Contains the `README.md` that appears on the [@Beatrix-lang](https://github.com/Beatrix-lang) organization profile.
- `.github`: (Future) Organization-wide issue templates, contribution guidelines, and automation workflows.

## 📌 Note
If you are looking for the actual **Beatrix compiler source code**, please head over to the [Beatrix Core](https://github.com/Beatrix-lang/beatrix) repository.
